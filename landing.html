Setup

Up and running in 3 minutes

If you already use Claude Code, you're 90% there.

1

Download Jents

Install the macOS app. It's an Electron app with native PTY support - no Docker, no VMs.

2

Edit team.json

Define your agents. Each one gets an ID, name, working directory, and command.

{
  "agents": [{
    "id": "ops",
    "name": "Operations",
    "cwd": "~/agents/ops",
    "command": "claude",
    "color": "#5b8def"
  }]
}
3

Launch

Your agents appear in the sidebar. Start them with Cmd+R. Jents auto-saves sessions and sends notifications from there.

Who It's For

Your agents deserve an org chart

★

Solo Founder

You're one person running a company with 5 Claude agents. Ops, Finance, Marketing, Product, Engineering. You need to see all of them at once and know the moment one needs input.

⚙

Technical Operator

You've already written CLAUDE.md files, connected MCP servers, and set permission modes. You just need a real app to manage the agents instead of 4 terminal tabs.

♠

Team Lead

You're building Claude Code workflows for your team. Jents gives each workflow a named container with its own config, logs, and notification stream.

The business world has spent centuries evolving containers for work. CFO. CMO. Chief of Staff. Product Manager. These titles aren't arbitrary. Each one is sufficiently narrow in objectives to be focused, and sufficiently broad in capabilities to be useful.

That's what makes them perfect containers for AI agents. And Jents is built for the person who's figured that out.

You're a founder, operator, or team lead who's already using Claude Code. You've written CLAUDE.md files that define each agent's mandate. You've connected MCP servers so they can query your data, read your messages, update your tools. You've set permission modes so some agents run autonomously while others wait for approval. The agents work. The problem is managing them.

You've got four terminal tabs open and you keep losing track of which one is doing what. An agent finishes a 10-minute task in the background and you don't notice for an hour. You want to paste a complex prompt but the terminal mangles it line by line. The output is beautiful markdown buried under ANSI codes and spinner characters. You close a tab and the session is gone.

Jents fixes all of that. Your agents live in a sidebar with colored avatars and status dots - you can see at a glance who's running, who's idle, who needs input. Desktop notifications tell you the moment an agent finishes working. The composer lets you draft multi-paragraph prompts and send them atomically. Reader view strips the terminal noise and renders clean formatted output you can copy into a doc with one click. Every session is auto-saved so you can review what any agent did last Tuesday.

Now multiply that by five. An Ops agent scanning messages and drafting your morning brief. A Finance agent running weekly pulses and catching cost spikes. A Marketing agent auditing spend efficiency across channels. A Product agent triaging tickets. Each one with its own CLAUDE.md, its own MCP tools, its own permission mode - all running in parallel in one window.

It's like a beautiful orchestra once everything's in motion. And it dramatically expands the surface area of what any one person can do when it's customized for your business and your life.
