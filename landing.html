Run

Five agents, one window, total visibility

Every agent gets a color-coded sidebar slot with live status. When one finishes work in the background, you get a macOS notification. Click it and you're there.

  • ✓ Color-coded sidebar with running/stopped/error status dots
  • ✓ Unread badges when background agents produce output
  • ✓ Smart notifications after 8s idle on substantial output
  • ✓ Permission mode selector per agent (Default / Auto / YOLO)
  • ✓ Resume last session with one click (--continue)
  • ✓ Double-click safety on stop and clear actions

Get Started

Up and running in 3 minutes

If you already use Claude Code, you're 90% there.

1

Download Jents

Install the macOS app. Electron with native PTY support - no Docker, no VMs, no config files.

2

Pick a starter pack

Choose a pre-built team or add agents one at a time. Each one gets a working directory, CLAUDE.md, and run files.

3

Start with Cmd+R

Your agents appear in the sidebar. Start them, and Jents handles notifications, session logs, and file tracking from there.

Who It's For

Your agents deserve an org chart

🎯

Solo Founder

You're one person running a company with 5 Claude agents. Ops, Finance, Marketing, Product, Engineering. You need all of them visible and notified the moment one needs input.

⚙

Technical Operator

You've already written CLAUDE.md files, connected MCP servers, and set permission modes. You just need a real app to manage the agents instead of 4 terminal tabs.

👥

Team Lead

You're building Claude Code workflows for your team. Jents gives each workflow a named container with its own config, logs, integrations, and notification stream.

The business world has spent centuries evolving containers for work. CFO. CMO. Chief of Staff. Product Manager. These titles aren't arbitrary. Each one is sufficiently narrow in objectives to be focused, and sufficiently broad in capabilities to be useful.

That's what makes them perfect containers for AI agents. And Jents is built for the person who's figured that out.

You're a founder, operator, or team lead who's already using Claude Code. You've written CLAUDE.md files that define each agent's mandate. You've connected MCP servers so they can query your data, read your messages, update your tools. You've set permission modes so some agents run autonomously while others wait for approval. The agents work. The problem is managing them.

You've got four terminal tabs open and you keep losing track of which one is doing what. An agent finishes a 10-minute task in the background and you don't notice for an hour. You want to paste a complex prompt but the terminal mangles it line by line. The output is beautiful markdown buried under ANSI codes and spinner characters.

Jents fixes all of that. Your agents live in a sidebar with colored avatars and status dots. Desktop notifications tell you the moment an agent finishes. The notepad lets you save prompts and fire them at any agent with one click. Reader view strips the noise and renders clean formatted output you can copy into a doc. Every session is auto-saved so you can review what any agent did last Tuesday.

It's like a beautiful orchestra once everything's in motion. And it dramatically expands the surface area of what any one person can do when it's customized for your business and your life.
